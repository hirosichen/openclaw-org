## 什麼是代理？

在 OpenClaw 中，代理（Agent）是一個可以獨立執行任務的 AI 實體。每個代理都有自己的：

- **個性設定** — 定義回應風格
- **系統提示詞** — 指導行為的基礎指令
- **工具集** — 可以使用的功能
- **記憶** — 對話歷史和偏好

## 建立自訂代理

你可以在設定檔中定義自訂代理：

```yaml
# ~/.openclaw/agents/translator.yaml
name: 翻譯助理
description: 專門處理翻譯任務的代理
language: zh-TW
system_prompt: |
  你是一個專業的翻譯助理，擅長中英文互譯。
  請保持翻譯的準確性和自然度。
tools:
  - web_search
  - dictionary
```

## 啟用代理

```bash
# 列出所有代理
openclaw agent list

# 使用特定代理對話
openclaw chat --agent translator "Please translate this to Chinese"
```

## 代理生命週期

代理的生命週期包含以下階段：

1. **初始化** — 載入設定和記憶
2. **接收訊息** — 從使用者接收輸入
3. **處理** — 分析意圖並生成回應
4. **工具呼叫** — 如需要，執行工具
5. **回應** — 將結果傳送給使用者
6. **記憶更新** — 儲存重要的對話上下文

## 預設代理

OpenClaw 內建幾個實用的代理：

| 代理 | 用途 |
|------|------|
| `default` | 通用助理 |
| `coder` | 程式開發助理 |
| `writer` | 寫作助理 |
| `analyst` | 資料分析助理 |
