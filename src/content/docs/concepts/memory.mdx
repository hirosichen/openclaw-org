## 記憶系統概覽

OpenClaw 的記憶系統讓 AI 助理能夠記住使用者的偏好、對話歷史和重要資訊，提供更個人化的互動體驗。

## 記憶類型

### 短期記憶

短期記憶存儲當前對話的上下文，在對話結束後自動清除：

- 對話歷史
- 臨時變數
- 工具執行結果

### 長期記憶

長期記憶持久儲存在本機，跨對話保留：

- 使用者偏好
- 重要事實
- 學習到的模式

## 記憶設定

```yaml
# ~/.openclaw/config.yaml
memory:
  enabled: true
  short_term:
    max_tokens: 4096
  long_term:
    enabled: true
    storage: sqlite
    path: ~/.openclaw/memory.db
    max_entries: 10000
```

## 管理記憶

```bash
# 查看記憶統計
openclaw memory stats

# 搜尋記憶
openclaw memory search "使用者偏好"

# 清除短期記憶
openclaw memory clear --short-term

# 匯出記憶
openclaw memory export backup.json
```

## 隱私保護

所有記憶資料都儲存在你的本機裝置上：

- 不會上傳到雲端
- 使用加密儲存
- 可以隨時完全刪除
- 支援選擇性遺忘
