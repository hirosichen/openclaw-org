## Messenger 整合（實驗性）

透過 Facebook Messenger Platform 將 OpenClaw 連接到 Messenger。

## 前置需求

- Facebook Page
- Meta for Developers 帳號
- 已通過 App Review（用於正式環境）

## 設定步驟

### 1. 建立 Facebook App

1. 前往 [Meta for Developers](https://developers.facebook.com/)
2. 建立新的 App，選擇「Business」類型
3. 新增 Messenger 產品
4. 連結你的 Facebook Page

### 2. 取得 Token

生成 Page Access Token 並記下。

### 3. 設定 OpenClaw

```bash
openclaw channel enable messenger
openclaw channel config messenger \
  --page-access-token YOUR_PAGE_TOKEN \
  --verify-token YOUR_VERIFY_TOKEN
```

### 4. 設定 Webhook

在 Meta Developer Console 設定 Webhook，訂閱 `messages` 事件。

## 注意事項

- Messenger 整合目前為實驗性功能
- 正式環境需要通過 Meta 的 App Review
- 免費版有每日訊息數量限制

## 設定範例

```yaml
channels:
  messenger:
    enabled: true
    page_access_token: YOUR_TOKEN
    verify_token: YOUR_VERIFY_TOKEN
    webhook_path: /webhook/messenger
```
