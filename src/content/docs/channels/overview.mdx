## 頻道整合

OpenClaw 支援多種聊天頻道，讓你透過日常使用的通訊軟體與 AI 助理互動。

## 支援的頻道

| 頻道 | 狀態 | 特色 |
|------|------|------|
| [LINE](/docs/channels/line) | ✅ 穩定 | 台灣最普及的通訊軟體 |
| [Telegram](/docs/channels/telegram) | ✅ 穩定 | 功能豐富、Bot API 完善 |
| [WhatsApp](/docs/channels/whatsapp) | ✅ 穩定 | 全球使用者最多 |
| [Discord](/docs/channels/discord) | ✅ 穩定 | 適合社群和團隊 |
| [Signal](/docs/channels/signal) | 🧪 實驗 | 最高隱私標準 |
| [Slack](/docs/channels/slack) | ✅ 穩定 | 企業團隊首選 |
| [Messenger](/docs/channels/messenger) | 🧪 實驗 | Facebook 生態系 |

## 頻道運作原理

每個頻道整合都透過以下流程運作：

1. **註冊 Bot** — 在目標平台建立 Bot 帳號
2. **設定 Webhook** — 將平台的 Webhook 指向 OpenClaw
3. **驗證連線** — 確認雙向通訊正常
4. **開始使用** — 透過聊天 App 與 AI 互動

## 快速設定

```bash
# 啟用頻道
openclaw channel enable line

# 設定頻道
openclaw channel config line --token YOUR_TOKEN

# 測試連線
openclaw channel test line
```

## 多頻道同時使用

OpenClaw 支援同時啟用多個頻道，所有頻道共享相同的代理和記憶系統。
