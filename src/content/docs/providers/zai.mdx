Z.AI 是 **GLM** 模型的 API 平台。它提供 REST API 並使用 API 金鑰進行驗證。在 OpenClaw 中透過 `zai` 供應商整合。

## CLI 設定

```bash
openclaw onboard --auth-choice zai-api-key
```

或非互動式：

```bash
openclaw onboard --non-interactive \
  --mode local \
  --auth-choice zai-api-key \
  --zai-api-key "$ZAI_API_KEY"
```

## 設定片段

```json5
{
  env: { ZAI_API_KEY: "sk-..." },
  agents: {
    defaults: {
      model: { primary: "zai/glm-5" },
    },
  },
}
```

## 技術細節

- **模型參考格式：** `zai/<model>`（例如 `zai/glm-5`）
- **驗證：** Bearer token，使用 API 金鑰
- **工具呼叫串流：** 預設啟用 `tool_stream`

### 停用工具呼叫串流

如果需要，可以在模型參數中停用 `tool_stream`：

```json5
{
  providers: {
    zai: {
      models: {
        "zai/glm-5": {
          params: { tool_stream: false },
        },
      },
    },
  },
}
```
