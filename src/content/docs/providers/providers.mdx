OpenClaw 可以使用多種 LLM 供應商。選擇一個供應商、進行驗證，然後將預設模型設定為 `provider/model`。

尋找聊天頻道文件（WhatsApp/Telegram/Discord/Slack/Mattermost（外掛）等）？請參閱 [頻道](/channels)。

## 精選：Venice（Venice AI）

Venice 是我們推薦的 Venice AI 設定，提供注重隱私的推理，並可選擇使用 Opus 處理困難任務。

- 預設：`venice/llama-3.3-70b`
- 最佳整體：`venice/claude-opus-45`（Opus 仍然是最強的）

參閱 [Venice AI](/docs/providers/venice)。

## 快速開始

1. 向供應商進行驗證（通常透過 `openclaw onboard`）。
2. 設定預設模型：

```json5
{
  agents: { defaults: { model: { primary: "anthropic/claude-opus-4-6" } } },
}
```
