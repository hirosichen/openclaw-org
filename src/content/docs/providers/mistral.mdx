OpenClaw 原生支援 Mistral，涵蓋多種功能：模型路由、語音轉錄和記憶嵌入。

## 驗證

驗證依賴 `MISTRAL_API_KEY` 環境變數。

### CLI 設定

```bash
openclaw onboard --auth-choice mistral-api-key
```

或非互動式：

```bash
openclaw onboard --non-interactive \
  --mode local \
  --auth-choice mistral-api-key \
  --mistral-api-key "$MISTRAL_API_KEY"
```

## 文字/圖片路由

使用 `mistral/...` 前綴存取 Mistral 的語言模型。預設模型為 `mistral-large-latest`。

### 設定片段

```json5
{
  env: { MISTRAL_API_KEY: "..." },
  agents: {
    defaults: {
      model: { primary: "mistral/mistral-large-latest" },
    },
  },
}
```

## 語音轉錄（Voxtral）

Voxtral 整合提供媒體理解能力，標準語音模型為 `voxtral-mini-latest`。

### 設定片段

```json5
{
  env: { MISTRAL_API_KEY: "..." },
  transcription: {
    provider: "mistral",
    model: "voxtral-mini-latest",
  },
}
```

**技術細節：**

- **API 基礎 URL：** `https://api.mistral.ai/v1`
- **轉錄端點：** `/v1/audio/transcriptions`

## 記憶嵌入

將 Mistral 設定為嵌入供應商，用於記憶搜尋功能。

### 設定片段

```json5
{
  env: { MISTRAL_API_KEY: "..." },
  memorySearch: {
    provider: "mistral",
    model: "mistral-embed",
  },
}
```

**技術細節：**

- **嵌入端點：** `/v1/embeddings`
- **預設模型：** `mistral-embed`

## 注意事項

- 所有功能均使用相同的 `MISTRAL_API_KEY`
- API 基礎 URL 預設為 `https://api.mistral.ai/v1`
- 如需更多可用模型資訊，請參閱 [Mistral 官方文件](https://docs.mistral.ai/)
