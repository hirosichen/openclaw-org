GLM 是透過 Z.AI 平台存取的模型家族。在 OpenClaw 中，這些模型使用 `zai` 供應商，模型識別碼格式為 `zai/glm-5`。

## CLI 設定

```bash
openclaw onboard --auth-choice zai-api-key
```

## 設定片段

```json5
{
  env: { ZAI_API_KEY: "sk-..." },
  agents: {
    defaults: {
      model: { primary: "zai/glm-5" },
    },
  },
}
```

## 可用模型

常見的模型 ID 包括：

| 模型 ID | 說明 |
| --- | --- |
| `glm-5` | 最新版本 |
| `glm-4.7` | GLM 4.7 版 |
| `glm-4.6` | GLM 4.6 版 |

> **提示：** 模型版本和可用性可能會變更，請查閱 Z.AI 官方文件以確認目前的模型列表。
