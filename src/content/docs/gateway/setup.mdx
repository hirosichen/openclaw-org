## 閘道器概覽

閘道器（Gateway）是 OpenClaw 接收和發送訊息的中央樞紐，負責處理所有頻道的通訊。

## 基本設定

```yaml
# ~/.openclaw/config.yaml
gateway:
  host: 0.0.0.0
  port: 3000
  cors:
    enabled: true
    origins:
      - http://localhost:*
```

## 啟動閘道器

```bash
# 啟動（含所有已啟用的頻道）
openclaw start

# 僅啟動閘道器
openclaw gateway start

# 指定連接埠
openclaw gateway start --port 8080
```

## Webhook 路由

閘道器自動為每個頻道建立 Webhook 端點：

| 頻道 | Webhook 路徑 |
|------|-------------|
| LINE | `/webhook/line` |
| Telegram | `/webhook/telegram` |
| WhatsApp | `/webhook/whatsapp` |
| Discord | `/webhook/discord` |
| Slack | `/webhook/slack` |

## 健康檢查

閘道器提供健康檢查端點：

```bash
curl http://localhost:3000/health
```

回應：

```json
{
  "status": "ok",
  "version": "1.0.0",
  "channels": {
    "line": "connected",
    "telegram": "connected"
  }
}
```

## 日誌設定

```yaml
gateway:
  logging:
    level: info    # debug, info, warn, error
    format: json   # json, text
    file: ~/.openclaw/logs/gateway.log
```
