## 基本設定

OpenClaw 使用 YAML 設定檔來管理所有選項。設定檔位於：

```
~/.openclaw/config.yaml
```

首次執行時，OpenClaw 會自動建立預設設定檔。

## 設定 AI 模型

OpenClaw 支援多種 AI 模型提供者。使用 CLI 快速設定：

```bash
# 使用 OpenAI
openclaw config set model openai:gpt-4
openclaw config set openai_api_key sk-your-key-here

# 使用 Anthropic
openclaw config set model anthropic:claude-3-opus
openclaw config set anthropic_api_key sk-ant-your-key-here

# 使用 Google Gemini
openclaw config set model google:gemini-pro
openclaw config set google_api_key your-key-here
```

## 設定檔範例

```yaml
# ~/.openclaw/config.yaml
model: openai:gpt-4
openai_api_key: sk-your-key-here

# 代理設定
agent:
  name: 小助理
  language: zh-TW
  personality: helpful

# 記憶設定
memory:
  enabled: true
  max_tokens: 4096
```

## 環境變數

你也可以使用環境變數來設定敏感資訊：

```bash
export OPENCLAW_MODEL=openai:gpt-4
export OPENAI_API_KEY=sk-your-key-here
```

環境變數的優先級高於設定檔。
