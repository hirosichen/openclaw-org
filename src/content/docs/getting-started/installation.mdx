## 系統需求

在安裝 OpenClaw 之前，請確認你的系統符合以下需求：

- **作業系統**：macOS 12+、Ubuntu 20.04+、Windows 10+
- **記憶體**：至少 4 GB RAM（建議 8 GB 以上）
- **磁碟空間**：至少 2 GB 可用空間
- **網路**：需要網路連線以下載模型和套件

## 使用安裝腳本

最簡單的安裝方式是使用官方安裝腳本：

```bash
curl -fsSL https://get.openclaw.org | bash
```

安裝腳本會自動偵測你的作業系統並安裝適當的版本。

## 使用套件管理器

### macOS (Homebrew)

```bash
brew install openclaw
```

### Ubuntu / Debian

```bash
sudo apt update
sudo apt install openclaw
```

### Windows (Scoop)

```powershell
scoop install openclaw
```

## 驗證安裝

安裝完成後，執行以下指令確認安裝成功：

```bash
openclaw --version
```

你應該會看到類似以下的輸出：

```
OpenClaw v1.0.0
```
