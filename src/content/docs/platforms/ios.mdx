## iOS 上使用 OpenClaw

OpenClaw 主要在你的電腦上運行，但你可以透過 iOS 裝置上的聊天 App 與它互動。

## 使用方式

### 透過聊天 App

最簡單的方式是透過 iOS 上已安裝的聊天 App 與 OpenClaw 互動：

1. **LINE** — 加入你的 OpenClaw Bot 好友
2. **Telegram** — 搜尋你的 Bot 帳號
3. **WhatsApp** — 加入 Bot 的電話號碼

### 透過 Web 介面

用 Safari 或 Chrome 造訪 OpenClaw 的 Web 介面：

```
http://your-computer-ip:3000
```

建議將頁面加入主畫面以獲得類似 App 的體驗。

## 設定遠端存取

要從 iOS 裝置存取家中的 OpenClaw，你需要：

1. 設定[遠端存取](/docs/gateway/remote-access)
2. 確保你的電腦在運行 OpenClaw 服務
3. 連接到相同的網路或使用隧道

## 捷徑整合

你可以使用 iOS 捷徑（Shortcuts）來快速與 OpenClaw 互動：

1. 打開「捷徑」App
2. 建立新捷徑
3. 新增「取得 URL 內容」動作
4. 設定你的 OpenClaw API 端點

## 注意事項

- OpenClaw 本身不在 iOS 上運行
- 需要你的電腦保持開機和運行 OpenClaw
- 建議設定穩定的網路連線
