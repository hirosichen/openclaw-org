## Discord 最新專案

- **PR Review → Telegram Feedback** (@bangnokia) — OpenCode 完成變更後開啟 PR，OpenClaw 審閱差異並在 Telegram 回覆建議與合併決定。
- **Wine Cellar Skill in Minutes** (@prades_maxime) — 請「Robby」建立一個本地酒窖技能。它要求匯出範例 CSV 與儲存位置，然後快速建置並測試技能（範例含 962 瓶酒）。
- **Tesco Shop Autopilot** (@marchattonhere) — 每週用餐計畫 → 常購品 → 預約配送時段 → 確認訂單。無需 API，純靠瀏覽器控制。
- **SNAG Screenshot-to-Markdown** (@am-will) — 按快捷鍵截取螢幕區域 → Gemini 視覺辨識 → 即時 Markdown 複製到剪貼簿。
- **Agents UI** (@kitze) — 桌面應用程式，管理跨 Agents、Claude、Codex 與 OpenClaw 的技能/指令。
- **Telegram Voice Notes** (papla.media) — 包裝 papla.media TTS 並以 Telegram 語音訊息傳送結果。
- **CodexMonitor** (@odrobnik) — 透過 Homebrew 安裝的輔助工具，列出/檢視/監控本地 OpenAI Codex 工作階段。
- **Bambu 3D Printer Control** (@tobiasbischoff) — 控制與排除 BambuLab 印表機問題：狀態、工作、攝影機、AMS、校準。
- **Vienna Transport** (@hjanuschka) — 維也納公共交通的即時出發時間、中斷通知、電梯狀態與路線規劃。
- **ParentPay School Meals** (@George5562) — 透過滑鼠座標自動化英國學校餐點 ParentPay 訂餐。
- **R2 Upload** (@julianengel) — 上傳至 Cloudflare R2/S3 並產生安全的預簽名下載連結。
- **iOS App via Telegram** (@coard) — 透過 Telegram 聊天建置完整的 iOS 應用程式（含地圖與錄音功能），並部署至 TestFlight。
- **Oura Ring Health Assistant** (@AS) — 整合 Oura 戒指資料與行事曆及預約的個人 AI 健康助手。
- **Kev's Dream Team** (@adam91holt) — 14+ 個代理在同一個 Gateway 下，由 Opus 4.5 協調器委派工作給 Codex 工作者。
- **Linear CLI** (@NessZerra) — 與代理工作流程整合的 Linear CLI。從終端管理議題、專案與工作流程。
- **Beeper CLI** (@jules) — 透過 Beeper Desktop 的本地 MCP API 讀取、傳送與封存訊息。

## 自動化與工作流程

- **Winix Air Purifier Control** (@antonplex) — Claude Code 發現了空氣清淨機控制介面，然後由 OpenClaw 管理室內空氣品質。
- **Pretty Sky Camera Shots** (@signalgaining) — 由屋頂攝影機觸發：每當天空好看時，OpenClaw 自動拍攝一張天空照片。
- **Visual Morning Briefing Scene** (@buddyhadry) — 排程提示每天早上產生一張「場景」圖片。
- **Padel Court Booking** (@joshp123) — Playtomic 場地空位查詢 + 訂場 CLI，絕不錯過任何空場。
- **Accounting Intake** — 從電子郵件收集 PDF，自動為稅務顧問準備文件。
- **Couch Potato Dev Mode** (@davekiss) — 一邊看 Netflix 一邊透過 Telegram 重建個人網站。
- **Job Search Agent** (@attol8) — 搜尋職缺、比對 CV 關鍵字、回傳相關機會。
- **Jira Skill Builder** (@jdrhyne) — OpenClaw 連接 Jira，即時產生新技能。
- **Todoist Skill via Telegram** (@iamsubhrajyoti) — 透過 OpenClaw 在聊天中產生技能來自動化 Todoist 任務。
- **TradingView Analysis** (@bheem1798) — 透過瀏覽器自動化登入 TradingView、截圖圖表、執行技術分析。
- **Slack Auto-Support** (@henrymascot) — 監控公司 Slack 頻道，自動回覆、轉發至 Telegram，並自主修復生產環境錯誤。
